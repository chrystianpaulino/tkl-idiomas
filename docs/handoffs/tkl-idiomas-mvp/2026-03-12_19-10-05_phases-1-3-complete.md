## Key Decisions

| Decision | Rationale | Alternatives Considered |
|----------|-----------|------------------------|
| `TurmaClass` model name (not `Class`) | `class` is a PHP reserved word | Using `Turma`, keeping `Class` with alias |
| `lessons.package_id` → `restrictOnDelete()` | Lessons are audit records; voiding a package must not destroy history | cascadeOnDelete (destructive), nullOnDelete (orphaned records) |
| `conducted_at` nullable | Lesson may be recorded with current time, but nullable prevents crashes | NOT NULL (causes issues if app forgets to supply it) |
| `role` NOT in `$fillable` | Privilege escalation prevention | In $fillable (risky), guarded property |
| `purchased_at` → `useCurrent()` | Package purchase timestamp should always be "now" | Required field with no default (fragile) |
